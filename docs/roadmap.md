# Roadmap

## Phase 1: Foundation

- finalize source shortlist
- finalize normalized schema
- build MVP backend folder
- build first 2-3 parsers
- store opportunities in database

## Phase 2: Applicant Workflow

- build frontend list page
- build opportunity detail page
- add `Applied` state
- add applied page/tab
- add visual applied badge on the main list

## Phase 3: Better Parsing

- add required-documents extraction
- add supervisor profile extraction
- add funding and stipend normalization
- add domain classification for AI / ML / DL / Agents / RAG

## Phase 4: Verification Layer

- official vs aggregator verification
- stale listing cleanup
- last-seen tracking
- closing-soon alerts

## Phase 5: Gradual Expansion

- add more countries
- add more official university/institute sources
- add notes, reminders, and document readiness tracking
